## Preliminary findings

### Participants' behaviour was categorised into following units by **data-driven, unsupervised learning**.
- stopping
- stopping and looking-around
- straight walking
- other, i.e., transition phase of above

### The behavioural data consists of:
- location (x,y,z)
- head orientation (x,y,z)
- 3D isovist (Embodied 3D Isovist, Krukar et al., 2021) metrics
